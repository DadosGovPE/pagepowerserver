Power BI Report Server

Acesso e Suporte

1. Deseja obter um login e senha pela 1ª vez

  • Peça ao seu núcleo setorial de informática (NSI) para providenciar junto à DGA/ATI.
  • Caso sua secretaria não possua NSI, envie e-mail para lista-pcad@ati.pe.gov.br para mais informações.

2. Já tem o login e deseja reset de senha

  • Envio de E-mail para csati@ati.pe.gov.br, intitulado "Power BI Report Server - Reset de Senha". Informar nome completo e e-mail institucional (@seplag.pe.gov.br).
  • Após o atendimento da solicitação, a ATI envia e-mail com senha temporária para o endereço institucional. Nele, seguem as instruções para alteração de senha.
  • Após alterar senha no link informado pela ATI, o acesso ao Report Server fica liberado com o uso da nova senha.

3. Caso você já possua login e senha e não consegue logar

  • Possivelmente sua senha expirou, clique no link: Alterar Senha Power BI.
  • Uma vez no site de alterar senha use powerbi\ antes do nome de usuário e insira a senha atual. Clique em "click here".
  • Crie uma nova senha com os requisitos de complexidade exigidos.
  • Tente acessar novamente em PowerBi

4. É de uma secretaria externa à SEPLAG e deseja acessar o painel de Execução Orçamentária

  • Formatar termo de responsabilidade sobre a utilização dos dados baseado nesse documento.
  • Enviar SEI para Instituto de Gestão (IG/SEPLAG).
  • Repassar seu login do Power BI Report Server para ncd.igpe@gmail.com junto com o número do SEI enviado.

5. Ainda em dúvidas?

  • Visite a wiki Servidor de Relatorios do Power BI
  • Envie um E-mail para lista-pcad@ati.pe.gov.br
