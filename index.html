1. Obter login e senha pela 1ª vez

  • Peça ao seu núcleo setorial de informática (NSI) para providenciar junto à DGA/ATI.
  • Caso sua secretaria não possua NSI, envie e-mail para lista-pcad@ati.pe.gov.br.

2. Resetar senha do Power BI

  • Envie e-mail para csati@ati.pe.gov.br com nome completo e e-mail institucional.
  • Você receberá uma senha temporária com instruções para troca.
  • Após alterar a senha, o acesso será liberado.

3. Já tem senha mas não consegue logar

  • Alterar Senha Power BI.
  • Use powerbi\ antes do nome de usuário e sua senha atual.
  • Crie nova senha com os requisitos exigidos.

4. Acesso para secretarias externas à SEPLAG

  • Formatar termo de responsabilidade com base no modelo apropriado.
  • Enviar SEI para o IGPE.
  • Repassar login + número do SEI para ncd.igpe@gmail.com.
